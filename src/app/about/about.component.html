À propos

Ce site est une application interactive qui vous permet de découvrir de manière aléatoire une variété de trois cocktails à la fois, leurs ingrédients, et les étapes de préparation. Que vous soyez amateur ou passionné, vous trouverez forcément un cocktail à votre goût.

L’objectif principal de ce projet est d’explorer Angular et de mettre en pratique le développement d'une application moderne, responsive et agréable à utiliser.

Les données proviennent de l’API ouverte TheCocktailDB , que nous remercions pour la richesse de leur base.

Technologies utilisées

  • Angular
  • Tailwind CSS
  • TypeScript
  • API REST

Merci de visiter notre site ! Nous espérons que vous apprécierez votre expérience.
